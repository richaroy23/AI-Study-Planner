# 🤖 AI Study Planner

An intelligent web application that helps students plan their study schedule using AI.

---

## 🚀 Features

* Add subjects and deadlines
* Generate AI-based study plans
* Simple and clean user interface
* Backend powered by Django
* Data stored using MySQL (planned)
* AI integration using OpenAI API (planned)

---

**Frontend:**

* HTML
* CSS

**Backend:**

* Django (Python)

* MySQL (currently using SQLite for development)

## ⚙️ Setup Instructions

### 1. Clone the repository

```bash
git clone <your-repo-link>
cd AI-Study-Planner
```

### 2. Create virtual environment

```bash
python -m venv venv
```

### 3. Activate virtual environment

**PowerShell:**

```bash
.\venv\Scripts\Activate.ps1
```

### 4. Install dependencies

```bash
pip install django
```

### 5. Run migrations

```bash
python manage.py migrate
```

### 6. Start server

```bash
python manage.py runserver
```
